## Functions

### サインイン/サインアップ

ユーザは**Email**でアカウント作成が可能。

<br />

### タスク追加 / 完了 / 削除 / 未完了へ戻す

- タスクを追加・完了・削除・未完了へ戻す。
- 登録できる ToDo は 5 個まで（5 個登録すると Input が disabled になる）。
- 同名 ToDo は登録不可。
